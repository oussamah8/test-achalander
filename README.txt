Pour éxecuter le programme, veuillez :

- Ouvrir une invite de commande dans le dossier "tri-app" et executer la commande "npm install" pour installer les "dependencies" utilisés.
- Executer la commande "node server.js" qui va commencer le serveur dans le port 4000.
- Ouvrir une invite de commande dans le dossier "tri-app/client" et executer la commande "npm install" pour installer les "dependencies" utilisés.
- Executer la commande "npm start" pour que l'application puisse démarrer avec le port 3000, donc dans le navigateur le lien sera "http://localhost:3000"

Pour utiliser l'application :
- Vous trouverez un champ déjà créé pour le remplir avec un numéro quelconque.
- Le boutton "+" au dessous c'est pour créer un nouveau champ pour introduire un autre numéro.
- On peut ajouter autant que besoin.
- Après on a le boutton "TRIER !" mais avant de le cliquer on doit remarquer qu'il y a un boutton au dessus pour switcher entre un ordre croissant ou décroissant de tri.
- Lorsqu'on clique sur le boutton "TRIER !" on peut voir le résultat de tri à droite.

PS: Pour l'algorithme de tri, j'ai implémenté l'algorithme d'insertion (Insertion Sort), il y en a plein d'autres. C'est pas le meilleur algorithme mais c'est pas mauvais, surtout quand on n'a pas un grand nombre de numéros à trier, et sa complexité est moyenne.
